# Presentation Speaker Notes

Este documento descreve o skill `presentation-speaker-notes`.

## Descrição

Gera um roteiro detalhado do apresentador para cada slide: pontos de conversa, marcações de timing, frases de transição e uma seção de Q&A com perguntas prováveis do público e respostas sugeridas.

## Ativadores

`escreva as notas do apresentador`, `crie um roteiro de apresentação`, `prepare o guia do apresentador`, `preparação para Q&A`, `timing dos meus slides`

## Saída por Slide

- Timing + tempo decorrido acumulado
- Sinal de avanço (clique / pausa / revelar)
- Roteiro em linguagem natural (2–4 frases)
- Frase de ênfase chave
- Transição para o próximo slide

## Seção Q&A

- 5–8 perguntas prováveis com respostas sugeridas
- Enquadramento diplomático para tópicos sensíveis
- Frases de desvio para perguntas fora do escopo
