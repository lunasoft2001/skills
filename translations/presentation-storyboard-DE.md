# Presentation Storyboard

Dieses Dokument beschreibt den Skill `presentation-storyboard`.

## Beschreibung

Strukturiert den narrativen Bogen einer Präsentation Folie für Folie. Für jede Folie werden eine Kernbotschaft, ein Ziel, eine empfohlene Dauer und ein visueller Vorschlag erstellt. Verwendet einen 3-Akt-Bogen (Kontext / Inhalt / Aktion) und die Ein-Botschaft-pro-Folie-Regel.

## Auslöser

`Folien planen`, `Präsentation strukturieren`, `Deck-Gliederung erstellen`, `narrativen Bogen definieren`, `Storyboard für Präsentation`

## Ausgabe pro Folie

- **Titel** — Folienüberschrift
- **Botschaft** — einziger Kernpunkt
- **Ziel** — was das Publikum nach dieser Folie fühlt
- **Dauer** — zugeteilte Zeit
- **Visueller Vorschlag** — Diagramm / Foto / Text
- **Übergang** — Verbindungssatz zur nächsten Folie

## Narrative Modelle

- Standard-Business (3 Akte)
- Pitch Deck (Investoren)
- Bildung / Workshop
- Executive Summary (max. 10 Min.)
