# Obsidian Markdown

Skill pour creer, normaliser et corriger des notes Markdown compatibles avec Obsidian.

## Objectif

Assurer un rendu correct dans Obsidian avec un frontmatter YAML valide, des taches checkbox, des callouts et des wikilinks.

## Structure

```text
obsidian-markdown/
  SKILL.md
  references/
    frontmatter-schema.md
    obsidian-callouts.md
    obsidian-task-conventions.md
    obsidian-linking-rules.md
    obsidian-validation-checklist.md
    templates-prd-handoff-meeting.md
```

## Fonctionnalites principales

- Validation stricte du frontmatter (pas de tabulations)
- Normalisation Markdown pour documents operationnels
- Standardisation des taches (`- [ ]`, `- [x]`)
- Modeles de callouts
- Liens internes avec wikilinks
- Modeles prets a l emploi (PRD, handoff, reunion)

## Cas d usage typiques

- Corriger un frontmatter casse dans Obsidian
- Convertir un markdown generique en format Obsidian
- Uniformiser des PRD quotidiens et des handoffs
- Ajouter le suivi operationnel avec taches et callouts
