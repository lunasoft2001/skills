# Presentation Speaker Notes

Questo documento descrive lo skill `presentation-speaker-notes`.

## Descrizione

Genera uno script dettagliato del relatore per ogni diapositiva: punti di discussione, segnali di timing, frasi di transizione e una sezione Q&A con domande probabili del pubblico e risposte suggerite.

## Trigger

`scrivi le note del relatore`, `crea un copione di presentazione`, `prepara la guida del presentatore`, `preparazione Q&A`, `timing delle mie diapositive`

## Output per Diapositiva

- Timing + tempo trascorso cumulativo
- Segnale di avanzamento (clic / pausa / rivela)
- Script in linguaggio naturale (2–4 frasi)
- Frase di enfasi chiave
- Transizione alla diapositiva successiva

## Sezione Q&A

- 5–8 domande probabili con risposte suggerite
- Formulazione diplomatica per argomenti sensibili
- Frasi di deviazione per domande fuori ambito

## Ritmo del Discorso

| Stile | Ritmo |
|-------|-------|
| Conversazionale | 120 ppm |
| Formale | 100 ppm |
| Storytelling | 110 ppm |

## File di Output

`/deliverables/<slug>/speaker-notes.docx`
