# Obsidian Markdown

Skill per creare, normalizzare e correggere note Markdown compatibili con Obsidian.

## Obiettivo

Garantire un rendering corretto in Obsidian con frontmatter YAML valido, task checkbox, callout e wikilink.

## Struttura

```text
obsidian-markdown/
  SKILL.md
  references/
    frontmatter-schema.md
    obsidian-callouts.md
    obsidian-task-conventions.md
    obsidian-linking-rules.md
    obsidian-validation-checklist.md
    templates-prd-handoff-meeting.md
```

## Funzionalita principali

- Validazione rigorosa del frontmatter (no tab)
- Normalizzazione markdown per documenti operativi
- Standard task checkbox (`- [ ]`, `- [x]`)
- Pattern callout per contesto e rischi
- Collegamenti interni con wikilink
- Template pronti (PRD, handoff, meeting)

## Casi d uso tipici

- Correggere frontmatter non valido in Obsidian
- Convertire markdown generico in formato Obsidian
- Uniformare PRD giornalieri e handoff
- Aggiungere tracciamento operativo con task e callout
