# Obsidian Markdown

Skill to create, normalize, and repair Obsidian-ready Markdown notes.

## Purpose

Ensure notes render correctly in Obsidian using valid YAML Properties, checkbox task blocks, callouts, and wikilinks.

## Structure

```text
obsidian-markdown/
  SKILL.md
  references/
    frontmatter-schema.md
    obsidian-callouts.md
    obsidian-task-conventions.md
    obsidian-linking-rules.md
    obsidian-validation-checklist.md
    templates-prd-handoff-meeting.md
```

## Main Features

- Strict frontmatter validation (no tabs, valid list indentation)
- Markdown normalization for operational documents
- Task checkbox standardization (`- [ ]`, `- [x]`)
- Obsidian callout patterns
- Internal linking with wikilinks
- Ready-to-use templates (PRD, handoff, meeting)

## Typical Use Cases

- Fix a broken Obsidian frontmatter block
- Convert generic markdown into Obsidian format
- Standardize daily PRDs and handoff docs
- Add operational task tracking and context callouts
