# Presentation Speaker Notes

Ce document décrit le skill `presentation-speaker-notes`.

## Description

Génère un script détaillé du présentateur pour chaque diapositive : points de discussion, repères de timing, phrases de transition et une section Q&R avec les questions probables du public et les réponses suggérées.

## Déclencheurs

`écrire les notes du présentateur`, `créer un script de présentation`, `préparer le guide du conférencier`, `préparation Q&R`, `timing de mes diapositives`

## Sortie par Diapositive

- Timing + temps écoulé cumulé
- Repère d'avancement (clic / pause / révéler)
- Script en langage naturel (2–4 phrases)
- Phrase d'emphase clé
- Transition vers la diapositive suivante

## Section Q&R

- 5–8 questions probables avec réponses suggérées
- Formulation diplomatique pour les sujets sensibles
- Phrases de déflexion pour les sujets hors portée

## Rythme de Parole

| Style | Rythme |
|-------|--------|
| Conversationnel | 120 mpm |
| Formel | 100 mpm |
| Storytelling | 110 mpm |
