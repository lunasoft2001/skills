# Obsidian Markdown

Skill para criar, normalizar e corrigir notas Markdown compativeis com Obsidian.

## Objetivo

Garantir renderizacao correta no Obsidian com frontmatter YAML valido, tarefas checkbox, callouts e wikilinks.

## Estrutura

```text
obsidian-markdown/
  SKILL.md
  references/
    frontmatter-schema.md
    obsidian-callouts.md
    obsidian-task-conventions.md
    obsidian-linking-rules.md
    obsidian-validation-checklist.md
    templates-prd-handoff-meeting.md
```

## Funcionalidades principais

- Validacao estrita de frontmatter (sem tabs)
- Normalizacao markdown para documentos operacionais
- Padrao de tarefas checkbox (`- [ ]`, `- [x]`)
- Padroes de callouts
- Links internos com wikilinks
- Templates prontos (PRD, handoff, reuniao)

## Casos de uso tipicos

- Corrigir frontmatter quebrado no Obsidian
- Converter markdown generico para formato Obsidian
- Padronizar PRDs diarios e handoffs
- Adicionar rastreabilidade operacional com tarefas e callouts
