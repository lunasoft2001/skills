## 1. Pirámide de agresividad

```
[MÁS TRACCIÓN / MÁS HOOK]
        Asym Solid
        Asym Hybrid
        Sym Solid
        Sym Hybrid
        Sym Pearl
        Urethane (especial)
        Plastic / Spare
[MENOS TRACCIÓN / MÁS RECTO]
```

Cada escalón equivale aproximadamente a 1–2 tablones de diferencia en punto de break.

## 2. Categorías de bola

### Asym Solid (máxima agresividad)
- **Cuándo**: Patrones largos (>42 ft), alto volumen, pistas muy oily
- **Shape**: Entrada muy temprana, arco continuo y fuerte, máximo flare
- **Superficie**: Siempre sanded (500–2000 grit)
- **Limitar en**: Patrones cortos, pistas secas, rev-dominant extremo

### Asym Hybrid
- **Cuándo**: Patrones medianos-largos (38–44 ft), transiciones moderadas
- **Shape**: Balance entre tracción y longitud, reacción definida en backend
- **Superficie**: 2000–3000 grit (o polish para más longitud)
- **Versatilidad**: Alta — funciona como primera bola en muchos torneos

### Sym Solid
- **Cuándo**: Patrones mediados-largos con más desgaste, pistas sintéticas, necesitas consistencia
- **Shape**: Arco continuo y predecible, menos agresivo que Asym
- **Superficie**: 1000–3000 grit
- **Ideal para**: Strokers, jugadores que buscan repetibilidad

### Sym Hybrid
- **Cuándo**: Patrones medianos (35–42 ft), quieres longitud + reacción definida
- **Shape**: Length + flip controlado
- **Superficie**: 3000 grit o polish ligero (1500–2000 + polish)

### Sym Pearl
- **Cuándo**: Patrones cortos-medianos (<40 ft), pistas con bastante fricción, segunda bola de ajuste
- **Shape**: Máxima longitud, snap fuerte en backend
- **Superficie**: 4000 grit + polish o high polish
- **Cuidado**: En patrones muy oily puede irse sin reaccionar

### Urethane
- **Cuándo**: Pistas muy secas, bowlers en transición quemada, quieres controlar el angulo
- **Shape**: Lento, continuo, sin snap — absorbe fricción
- **Superficie**: 2000–4000 grit (no pulir urethane en general)
- **Efecto secundario**: Altera el patrón de aceite para los que juegan detrás

### Plastic / Spare
- **Cuándo**: Spares rectos o casi rectos — especialmente el 10-pin (diestros) o 7-pin (zurdos)
- **Shape**: Completamente recto
- **Superficie**: Alta (polish) o media

## 3. Cover Stocks: tipos y comportamiento

| Tipo | Porosidad | Fricción | Longitud | Cuándo |
|---|---|---|---|---|
| Solid reactive | Alta | Alta | Corta-Media | Mucho aceite |
| Hybrid reactive | Media-Alta | Media-Alta | Media | Condiciones mixtas |
| Pearl reactive | Baja-Media | Media | Larga | Poca friction / quema |
| Urethane | Media | Media-Baja* | Media | Pistas secas, control |
| Plastic | Muy baja | Muy baja | Muy larga | Spares |

*Urethane absorbe la fricción de manera diferente — su flare es mínimo pero transita bien en condiciones secas.

## 4. Superficie: grit y polish

### Escala Abralon (de más áspero a más fino)
```
180 → 360 → 500 → 1000 → 2000 → 3000 → 4000 → Polish
```

| Superficie | Comportamiento | Cuándo |
|---|---|---|
| 180–500 grit | Máxima tracción, entrada muy temprana | Patrones extremos (≥45 ft), aceite muy alto |
| 1000–2000 grit | Buena tracción, versatil | Patrones largos/medianos, uso general |
| 2000–3000 grit | Balance length/hook | Patrones medianos, transición moderada |
| 3000–4000 grit | Más longitud, reacción de backend | Patrones cortos, pistas con fricción |
| 4000 + polish | Máxima longitud, snap en backend | Patrones cortos/alta fricción, speed-dominant |
| High polish | Casi sin tracción en aceite | Spare ball, patrones muy cortos |

**Regla práctica**: Cambiar la superficie es el ajuste más rápido y efectivo. Antes de cambiar de bola, prueba con superficie.

**Cuándo lijar (re-surface):**
- Cada 60–75 juegos aprox. en solid/hybrid
- Cada 50 juegos en pearl (se carga antes)
- La bola pierde tracción gradualmente → síntoma: antes hacía 4 tablones, ahora 2

## 5. Heurísticas de selección

### Por patrón de aceite:

| Patrón | Primera bola | Segunda bola |
|---|---|---|
| House (típico 35–40 ft) | Sym Solid o Asym Hybrid | Sym Pearl o Urethane |
| Sport (40–44 ft) | Asym Hybrid o Asym Solid | Sym Hybrid |
| Challenge (+44 ft, alto vol) | Asym Solid (sanded) | Asym Hybrid |
| Corona / Short (<33 ft) | Urethane o Sym Pearl | Plastic |

### Por clasificación del jugador:

| Jugador | Cover preferido | Superficie inicial |
|---|---|---|
| Speed-dominant | Solid (más tracción) | 500–1000 grit |
| Matched | Hybrid | 2000 grit |
| Rev-dominant | Pearl o Hybrid | 3000–4000 grit |
| Stroker | Sym Solid o Hybrid | 2000–3000 grit |
| Cranker | Sym Hybrid o Pearl | 3000–4000 grit |
| Two-hander | Sym Pearl o Hybrid | 3000+ grit o polish |

### Por transición de pista:

- **Pista fresca** (game 1–2): Usar bola que aguante la tracción (sanded solid/hybrid)
- **Pista media** (game 3–4): Pasar a hybrid pearl o ajustar superficie
- **Pista quemada** (game 5+): Pearl pulida, urethane, o ajustar línea de juego

## 6. Arsenal ideal por nivel

### Principiante / Amateur
1. House ball / plastic para spare
2. Una bola reactiva sym hybrid (all-around)

### Intermedio
1. Sym Solid (primera bola para aceite)
2. Sym Pearl (segunda bola para transición)
3. Plastic (spare ball)

### League player / avanzado
1. Asym Hybrid (primera bola heavy)
2. Sym Solid (segunda bola)
3. Sym Pearl (bola de ajuste / transición)
4. Urethane (opcional, transición quemada)
5. Plastic (spare recto)

### Competición / Torneo
1. Asym Solid (sanded, patrón largo)
2. Asym Hybrid (polished, versatil)
3. Sym Solid (backup / control)
4. Sym Pearl (pista seca / ajuste)
5. Urethane (situacional)
6. Plastic (spare)

## 7. Bola de spare

**Regla de oro**: Para hacer spares con consistencia, usa una bola **plástica** o muy débil y juega **recto** al spare. Elimina la variable del hook.

- Diestros: lanza al 10-pin desde el lado izquierdo, recto
- Zurdos: lanza al 7-pin desde el lado derecho, recto
- Para el 7-pin (diestros) o 10-pin (zurdos): bola reactiva puede funcionar, ajustar posición

**Superficie spare ball**: 4000 + high polish (mínima fricción).
