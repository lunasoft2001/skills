# Obsidian Callouts

Common callouts for operational notes.

## Templates

```markdown
> [!info] Context
> Short context summary.
```

```markdown
> [!todo] Pending
> Items still open.
```

```markdown
> [!warning] Risk
> Main risk and mitigation.
```

```markdown
> [!tip] Note
> Optional optimization or best practice.
```

## Guidance

1. Keep callouts short (2-5 lines).
2. Use one callout per intent (context, risk, todo).
3. Do not replace core content with only callouts.
