# Obsidian Validation Checklist

Run this checklist before closing edits.

## Frontmatter

- [ ] Starts at line 1
- [ ] Opening and closing `---` present
- [ ] No tabs in frontmatter
- [ ] `tags` is YAML list

## Body

- [ ] Headings are consistent
- [ ] At least one checkbox list for operational notes
- [ ] At least one callout when context/risk exists
- [ ] Internal references use wikilinks

## Render

- [ ] Obsidian Properties render correctly
- [ ] No red YAML block or parse warning
