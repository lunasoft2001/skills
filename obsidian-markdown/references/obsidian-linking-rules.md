# Obsidian Linking Rules

Use the right link type for each destination.

## Internal links (vault)

Use wikilinks:

- `[[README]]`
- `[[../DAILY_WORK/260403/README]]`
- `[[Proyecto X/Decision Log]]`

## External links

Use standard markdown links:

- `[Issue #10](https://github.com/owner/repo/issues/10)`
- `[Trello card](https://trello.com/c/xxxx)`

## Rules

1. Prefer wikilinks for local notes.
2. Use stable names for frequently linked notes.
3. Avoid broken relative paths when moving notes.
